Software Projects

The links below outline a few of my recent projects

Interplanetary Mission Design Module

This is a module written for Julia that allows for a host of different tools for planning of interplanetary missions. It can handle using lambert's algorithm to get from one planet to another at any time, and generate porkchop plots, etc.

  • GitHub

Kinematics Control Module

A module for python that allows for a wide range of rotational kinematics computations. Able to handle basic math in a wide variety of units, including Euler Angles, Principle Rotation Vectors, Quaternions, and Classical and Modified Rodriguez Parameters

  • GitHub

WheresMySat

A tool, currently in development, that allows for easy and restful orbit determination Kalman filtering intended to be a useful part of any satellite operator's toolbelt.
